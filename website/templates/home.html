Welcome to the DAB111 Group Project

This project integrates data collection, data processing, database storage, and web development into a single hands-on system. It demonstrates practical skills in Python programming, SQL operations, and Flask-based application development, fully aligning with all DAB111 course requirements.

Through this project, we work with real datasets, convert them into a structured SQLite database, and build a full website to explore and manage the data. The website allows:

  • Viewing summary information about the dataset
  • Displaying all records from the database
  • Uploading Excel files and inserting them into the SQLite database
  • Adding new records manually
  • Deleting records
  • Visualizing data through histograms

This system demonstrates the complete workflow from raw data → database → website.
