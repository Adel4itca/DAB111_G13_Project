Welcome to the DAB111 Group13 Project

This web application was developed as part of the DAB111 course to demonstrate the complete data workflow from raw data to a fully functional Flask-based system. The project integrates data collection, data cleaning, database management, and web development.

Using this system, you can perform the following tasks:

  • View summary information about the dataset
  • Display all book records stored in the SQLite database
  • Upload CSV files and automatically insert them into the database
  • Add new book records manually
  • Delete book records using BookID
  • Analyze category distribution through histograms

This platform provides a hands-on experience combining Python programming, SQL queries, Flask development, and data visualization.
