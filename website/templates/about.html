Project Purpose

This project was created to demonstrate the complete data analysis workflow covered in the DAB111 course. It includes collecting the dataset, cleaning and preparing the data, storing the processed data in a SQLite database, and developing a dynamic Flask application that allows users to interact with the data through CRUD operations, searching, and statistical visualization.
